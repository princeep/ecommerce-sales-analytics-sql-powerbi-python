# E-Commerce Sales & Profit Analysis (2024–2025)

## Project Overview

This project is built using live e-commerce sales data stored in CSV format and integrated through cloud storage.

The goal of this project is to analyze sales performance, customer behavior, profitability, regional performance, and business growth trends using SQL, Python, and Power BI.

## Tools & Technologies

* MySQL
* Python
* Power BI
* CSV Dataset
* OneDrive

## Business Objectives

This project answers important business questions such as:

* What is total revenue and profit?
* Which categories generate maximum sales?
* Which products are most profitable?
* Who are the top customers?
* Which regions perform best?
* What are monthly sales trends?
* Which payment methods are most used?

## Key Metrics

* Total Orders: 5000+
* Total Sales: 121M
* Total Profit: 20M
* Average Unit Price: 3.96K

## Business Insights

* Furniture generated the highest sales.
* North region showed the highest revenue contribution.
* COD and Credit Card were top payment methods.
* Q4 showed maximum sales growth.
